FlightGroup-specific mission goals

Remarks

Array is defined as [GoalIndex, {Argument, Condition, Amount, sbyte Points, bool Enabled, Team, bool Unknown10, bool Unknown11, bool Unknown12, Unknown13, bool Unknown14, Unknown15 (0), Unknown16}]. Note different types and Unknown15's value of zero
GoalIndex is 0-7.
Multiply Points by 250 to get actual value
