FlightGroup.Waypoints

Namespace: Idmr.Platform.Xvt
Assembly: Idmr.Platform.dll
Syntax: public short[,] Waypoints { get; }

Controls FlightGroup movement and locations. (Inherited from BaseFlightGroup)

Remarks

Array is defined as [{Start1, Start2, Start3, Start4, WP1, WP2, WP3, WP4, WP5, WP6, WP7, WP8, Rendezvous, Hyperspace, BRF1, BRF2, BRF3, BRF4, BRF5, BRF6, BRF7, BRF8}, {X, Y, Z, Enabled}]
Value stored is Coordinate * 160
Enabled is a boolean value, 0/1.
