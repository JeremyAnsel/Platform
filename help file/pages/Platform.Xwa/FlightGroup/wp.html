FlightGroup.Waypoints

Namespace: Idmr.Platform.Xwa
Assembly: Idmr.Platform.dll
Syntax: public short[,] Waypoints { get; }

Controls FlightGroup movement and locations. (Inherited from BaseFlightGroup)

Remarks

Array is defined as [{Start1, Start2, Start3, Hyperspace, R1O1WP1, R1O1WP2... R1O1WP8, R1O2WP1... R1O4WP8, R2O4WP1... R4O4WP8}, {X, Y, Z, Enabled, Region}]
Value stored is Coordinate * 160
Size of array is [132,5]. Eight WPs per order, four orders per region, four regions.
Enabled is a boolean value, 0/1.
