BaseFlightGroup.Roll

Namespace: Idmr.Platform
Assembly: Idmr.Platform.dll
Syntax: public short Roll { get; set; }

Craft x-axis orientation at start in degrees, -180 to 179.

Remarks

Calculated after Yaw and Pitch, in TIE only applies to SpaceObjects.
File stores value as sbyte.
