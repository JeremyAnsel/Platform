BaseFlightGroup.Yaw

Namespace: Idmr.Platform
Assembly: Idmr.Platform.dll
Syntax: public short Yaw { get; set; }

Craft z-axis orientation at start in degrees, -180 to 179.

Remarks

Calculated before Pitch and Roll, in TIE only applies to SpaceObjects.
File stores value as sbyte.
