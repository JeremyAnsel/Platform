Platform Namespace

The Platform namespace provides types from which all X-wing series mission classes are built.

Namespaces

NameDescription
TieProvides types to manipulate TIE95 mission files
XvtProvides types to manipulate XvT and BoP mission files
XwaProvides types to manipulate XWA mission files

Classes

NameDescription
BaseBriefingBase class for Briefings
BaseFlightGroupBase class for flightgroups
BaseGlobalsBase class for global goals
BaseMessageBase class for in-flight messages
BaseStringsBase class for string arrays
FixedSizeCollectionCollection class for fixed-size arrays such as Goals
MissionFileBase class for mission files
ResizableCollectionCollection class for variable-size arrays such as FlightGroups
