public static void CheckTarget(byte type, ref byte variable, out string errorMessage)
Checks Trigger.Type/Variable or Order.TargetType/Target pairs for values compatible with TIE. First checks for invalid Types, then runs through allows values for each Type. Does not verify FlightGroup, CraftWhen, GlobalGroup or GlobalUnit
