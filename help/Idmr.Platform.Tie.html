filePath - Full path to the unopened file
stream - Opened FileStream to the mission file

Methods

public static void CheckTarget(byte type, ref byte variable, out string errorMessage)
Checks Trigger.Type/Variable or Order.TargetType/Target pairs for values compatible with TIE. First checks for invalid Types, then runs through allows values for each Type. Does not verify FlightGroup, CraftWhen, GlobalGroup or Misc
type - Trigger.Type or Order.TargetType
variable - Trigger.Variable or Order.Target, may be updated
errorMessage - Error description if found, otherwise ""

public static byte CraftCheckbyte craftType)
Checks a CraftType for valid values and adjusts if necessary. Returns 255 if CraftType cannot be converted
craftType - The craft index to check

public void LoadFromFile(string filePath)
Loads a mission from a file. Calls LoadFromStream( )
filePath - Full path to the unopened file
Exceptions
System.IO.FileNotFoundException - filePath does not exist
System.IO.InvalidDataException - filePath is not a TIE mission file

public void LoadFromStream(System.IO.FileStream stream)
Loads a mission from an open FileStream
stream - Opened FileStream to the mission file
Exceptions
